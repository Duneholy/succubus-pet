# Contributing to Succubus Pet

First off, thank you for considering contributing to Succubus Pet!

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](https://github.com/Duneholy/succubus-pet/issues) page to see if someone else has already created a ticket. If not, go ahead and make one!

## Fork & create a pull request

If you'd like to contribute code:
1. Fork the repository.
2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## Coding Style
- Keep the code clean and well-documented.
- Ensure that the application runs cleanly on Windows with standard Python 3 installations.
